buildMocks ❌

A function that allows us to generate data for a specific Object Type

buildMocks(type, scenario);

Useful when we need to create a new node in a mutation. Accepts two arguments:

#type (Required)

String The type of the node we want to generate data for.

#scenario (Optional)

A specific scenario relevant to the current generation of the node. Read more about what they are in the Scenario section.
