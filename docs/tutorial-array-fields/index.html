Array Fields

Define the shape of Array Fields

Lets amend the schema to add an Airport type.

type City {
id: ID!
name: String
airports: [Airport]
}
type Airport {
id: ID!
name: String
}
# ...

Lets say we want our mock server to return two airports for our city, and the first airport be called Avram Iancu. To do that, we can use a Scenario for our field.

// ...
const typeBuilders = {
['City']: () => ({
name: casual.city,
airports: [{ name: 'Avram Iancu' }, {}],
}),
};
// ..

Running a city query will predictably return two airports:

{
"data": {
"city": {
"id": "m43g3ealt",
"name": "Cloog",
"population": 476,
"airports": [
{
"name": "Avram Iancu"
},
{
"name": "GENERATED_STRING"
}
]
}
}
}
